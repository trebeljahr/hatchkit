## Questions And Bugs

Open a GitHub issue for reproducible bugs, docs gaps, or feature requests.

Useful details:

- Command run.
- Current working directory.
- `hatchkit --version`.
- Relevant output from `hatchkit status --json`.
- Relevant output from `hatchkit doctor --json` for provider issues.

Do not paste secrets, tokens, dotenvx private keys, Coolify webhook URLs, or provider API responses containing credentials.
